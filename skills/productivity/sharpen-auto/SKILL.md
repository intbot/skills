---
name: sharpen-auto
description: Auto-firing variant of /sharpen. Use when the user pastes a rough or
  confusing prompt and wants it rewritten, clarified, or sharpened for an AI
  agent — even without the slash command.
---

Run the same routine as the **`sharpen`** skill: rewrite the user's rough prompt
into a clear agent prompt, show a **"What I changed:"** line, and **confirm via
AskUserQuestion before they send it** (✅ Use this · ✏️ Tighten further · ➕ Add
missing context · 💬 Chat about what I meant). On **Use this**, print the final
prompt as a clean, paste-ready block. Follow the `sharpen` skill's steps for the
full rules.
